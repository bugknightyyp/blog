# typescript 修饰符
data: [2016-11-11]
tags: [typescript] [modifier]

typescript 默认的修饰符是 `public`, `private` `protected`的作用和Java c#类似，
`private` 只能在本类里使用，不能被继承，`protected` 可以在被类和子类里使用，也就是可被继承。
