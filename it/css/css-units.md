# css里的单位
tags:[css] [unit]

css 有两种长度单位：相对单位和绝对单位。



##em
`em` is relative to the font-size of its direct or nearest parent
##rem
`rem`is only relative to the html (root) font-size.


## tips
空格不能出现在数字与单位之间，然而不过值是0，那么单位是不能省略的；

em 是相对于父元素，rem是相对于根元素；
